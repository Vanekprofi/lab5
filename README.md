## Задание 1
### Задача 1
#### Текст задачи
**Шаблоны. Дробь.**
В класс Дробь добавить интерфейс на два метода: получение вещественного 
значения, установка числителя и установка знаменателя. Сгенерировать такую 
версию дроби, которая будет кэшировать вычисление вещественного значения. 
Создать сущность Дробь со следующими особенностями:
- Имеет числитель: целое число
- Имеет знаменатель: целое число
- Дробь может быть создана с указанием числителя и знаменателя
- Может вернуть строковое представление вида "числитель/знаменатель"
- Необходимо корректно обрабатывать отрицательные значения
- Знаменатель не может быть отрицательным
- Переопределите метод сравнения объектов по состоянию

#### Алгоритм решения
Создание класса `Fraction` с приватными полями (числитель, знаменатель). 
Реализация интерфейса `FractionInterface` с методами `getRealValue()`, 
`setNumerator()`, `setDenominator()`. Проверка знаменателя на ноль и 
отрицательность в конструкторе и сеттерах. Создание класса `CachedFraction`, 
наследующего `Fraction`, с кэшированием вещественного значения. Кэш 
сбрасывается при изменении числителя или знаменателя. Переопределение 
методов `equals()`, `hashCode()`, `toString()`.

## Задание 2
### Задача 1
#### Текст задачи
**Структурные шаблоны. Количество мяуканий.**
Необходимо воспользоваться классом Кот и методом, принимающим всех 
мяукающих. Передать кота в указанный метод так, чтобы после окончания его 
работы узнать, сколько раз мяукал кот за время его работы. Кота изменять 
нельзя. Создать сущность Кот:
- Имеет Имя (строка)
- Для создания необходимо указать имя кота
- Может быть приведен к текстовой форме вида: "кот: Имя"
- Может помяукать с выводом текста: "Имя: мяу!"

Создать интерфейс Мяуканье с методом `public void meow()`.

#### Алгоритм решения
Создание интерфейса `Meowable` с методом `meow()`. Создание класса `Cat`, 
реализующего `Meowable`, с приватным полем `name`. Реализация декоратора 
`CountingCatDecorator`, который оборачивает `Cat` и подсчитывает количество 
вызовов `meow()`. Декоратор хранит ссылку на оригинального кота и счётчик 
мяуканий. Метод `getMeowCount()` возвращает количество вызовов.

## Задание 4
### Задача 4
#### Текст задачи
**Мап. Абитуриенты.**
В вузе абитуриенты проходят предварительное тестирование по двум предметам 
(от 0 до 100 баллов). К сдаче экзаменов допускаются абитуриенты, набравшие 
не менее 30 баллов по каждому предмету. На вход подаются сведения о 
результатах тестирования (до 500 участников). Формат ввода:
`<Фамилия> <Имя> <Баллы>`
Написать программу, выводящую фамилии и имена абитуриентов, не допущенных 
к сдаче экзаменов.

#### Алгоритм решения
Создание класса `Applicants` с внутренним классом `Applicant`. Поля: 
фамилия, имя, два балла. Метод `isAdmitted()` проверяет условие (>= 30 по 
каждому предмету). Использование `Map<String, Applicant>` для хранения 
данных. Метод `findFailed()` возвращает список не допущенных абитуриентов. 
В `Main` создаются тестовые данные и выводятся результаты.

## Задание 5
### Задача 5
#### Текст задачи
**Сет. Звонкие согласные.**
Файл содержит текст на русском языке. Напечатать в алфавитном порядке все 
звонкие согласные буквы, которые входят более чем в одно слово.

#### Алгоритм решения
Создание класса `VoicedConsonants` с набором звонких согласных 
(б, в, г, д, ж, з, й, л, м, н, р). Метод `findVoicedConsonants()` принимает 
текст, разбивает на слова. Для каждого слова определяется набор звонких 
согласных. Используется `Map<Character, Set<Integer>>` для отслеживания, 
в каких словах встречается каждая буква. В результат попадают буквы, 
встречающиеся более чем в одном слове. Сортировка по алфавиту.

## Задание 7
### Задача 1
#### Текст задачи
**Стрим. Точки.**
Дан набор объектов типа Point. Необходимо взять все Point в разных 
координатах, убрать с одинаковыми X,Y, отсортировать по X, отрицательные Y 
сделать положительными и собрать это все в ломаную (объект типа Polyline).

Создать классы:
- **Point**: координаты X, Y, текстовое представление "{X;Y}"
- **Line**: начало и конец (Point), представление "Линия от {X1;Y1} до {X2;Y2}"
- **Polyline**: массив точек, представление "Линия [Т1,T2,…,TN]"

#### Алгоритм решения
Создание классов `Point`, `Line`, `Polyline` в пакете `geometry`. Создание 
класса `PointStreamProcessor` с методом `processPoints()`. Использование 
Stream API: `distinct()` для удаления дубликатов, `sorted()` для сортировки 
по X, `map()` для модуля Y, `toArray()` для сбора в массив. Результат 
упаковывается в `Polyline`.

## Задание 7
### Задача 2
#### Текст задачи
**Стрим. Имена.**
Дан текстовый файл со строками, содержащими имя человека и его номер:
Вася:5
Петя:3
Аня:5
Номера могут повторяться. У некоторых может не быть номера. Написать стрим: 
читаются все люди из файла, имена приводятся к нижнему регистру с первой 
буквой в верхнем регистре.

#### Алгоритм решения
Создание класса `NameStreamProcessor` с методом `processFile()`. Чтение 
файла через `Files.lines()`. Фильтрация пустых строк. Разбиение по ":". 
Нормализация имени (первая буква заглавная, остальные строчные). Парсинг 
номера (если есть). Группировка по имени с помощью `Collectors.groupingBy()`. 
Результат: `Map<String, List<Integer>>`. Обработка `IOException`.
